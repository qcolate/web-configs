# Qcolate Babel Configs

[![version][npm-img]][npm-url] [![license][license-img]][license-url]


## Installation

```bash
yarn add -D @qcolate/babel-preset
```


## Usage

```javascript
// babel.config.js
module.exports = {
  presets: ['@qcolate/qcolate'],
};
```
