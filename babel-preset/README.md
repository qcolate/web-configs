# Qcolate Babel Configs

[![VERSION][version-badge-svg]][version-badge-url] [![LICENSE][license-badge-svg]](./LICENSE)


## Installation

```bash
yarn add -D @qcolate/babel-preset
```


## Usage

```javascript
// babel.config.js
module.exports = {
  presets: ['@qcolate/qcolate'],
};
```
